Q.What is an API?
Ans:-An API (Application Programming Interface) is a way for one app to talk to another.

Ex:-Your website → asks for data
Server → sends data
API → medium between them

Q.What is JSON?
 Ans:-JSON (JavaScript Object Notation) is a format to send data.
 It looks like JavaScript objects.
 Ex:-
 {
  "id": 1,
  "name": "Rahul",
  "email": "rahul@gmail.com"
}
//Key Points:
 Lightweight
 Easy to read
 Used in APIs

 Q.What is Fetch API?
 Ans:-fetch() is a JavaScript method to get or send data from/to server.
 
 Flow Diagram:fetch() → server → response → JSON → data

 Basic Syntax:-
 fetch(url)
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.log(error));

  url="https://jsonplaceholder.typicode.com/users"

  Ex:-
  fetch("https://jsonplaceholder.typicode.com/users")
  .then(res => res.json())
  .then(data => {
    console.log(data);
  })
  .catch(err => {
    console.log("Error:", err);
  });
--------------------------------------------Imp
//Fetch with async/await (Modern JS):-
async function getUsers() {
  try {
    let res = await fetch("https://jsonplaceholder.typicode.com/users");

    let data = await res.json();

    console.log(data);

  } catch (err) {
    console.log("Error:", err);
  }
}
getUsers();

Q.What is Priority Queue in JavaScript?
A queue where important tasks are executed first.
It is also informally called the Priority Queue.

Ex:-
console.log("Start");

fetch(url).then(() => {
  console.log("Fetched");
});

setTimeout(() => {
  console.log("Timer");
}, 0);
console.log("End");
